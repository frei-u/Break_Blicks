# ブロック崩し(Break Brocks)
## 概要
Siv3dで実装したブロック崩しゲームのMain関数部分のみを置いた。
実行ファイルは別のサイトに置く予定です。

## 工夫した点
難易度を3段階で設定した。

各難易度では、パドルの長さとボールのスピードを変更した。
